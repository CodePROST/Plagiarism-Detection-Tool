Plagiarism Detection Tool

The goal of this project is to detect plagiarism based on citation analysis.

View On GitHub
Downloads:
  • ZIP
  • TAR

    Description

    The application extracts the order of citations from the two documents, performs pattern matching between the sequences and returns the result. Also, for one of the shared references, it extracts the sentences cited with that reference from both the documents and quantifies the plagiarism with respect to that reference.

    How to run the application

    • Copy the 'Application' folder to the htdocs of your localhost.

    • Open web browser and enter the url as "http://localhost/Application.

    • Intro splash page opens up.

    • The next page corresponds to step one where two files of .txt format can be uploaded.

    • Once the files get successfully uploaded, just press the 'continue' buttons, you are done!!!

    2. PHP v.5.2.0+
